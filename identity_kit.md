## Part 1: Visual Identity Kit

### 1. Typography (Fonts)
*   **Heading Font:** `Inter` (sans-serif) — Clean, geometric, and modern. Emphasizes legibility and structure.
*   **Body & Code Font:** `JetBrains Mono` (monospace) — Used for subheads, tags, and data stats. Emphasizes my technical, ML engineering focus.

### 2. Color Palette
My palette is a modern dark-mode scheme, designed to be quiet enough to let my ML metrics and running AI agent be the loudest elements on the page.

*   **Primary Background:** `#0a0e17` (Deep Midnight Blue)
*   **Secondary Background (Cards):** `#111827` (Near-Black Slate)
*   **Primary Text:** `#e8ecf1` (Near-White Silver)
*   **Muted Text:** `#8b95a8` (Slate Gray)
*   **Accent Color:** `#6366f1` (Indigo Blue) — Used sparingly for links, active states, and highlights.

### 3. Monogram Logo & Favicon
My logo is a clean, code-inspired monogram set in my heading font:
```text
  { H }
```
It represents "Huzaifah" enclosed in code brackets, signaling my focus on coding and programmatic problem-solving.
